### Working with mjml in VSC

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM mjml
   ```sh
   npm install
   ```

### Visualize mjml in VSC


1. Install extensions ctrl +shift + p 
   * mjml open preview to the side
   * mjml copy html
